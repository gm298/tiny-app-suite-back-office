# Tiny Back Office — Sales Landing Page

Static marketing site (dark theme, matches the operations manual).

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
npx serve Sales-Landing-page
```

## GitHub Pages

Published at:

**https://gm298.github.io/tiny-app-suite-back-office/landing/**

From the repo root (requires `deploy-site` clone — see `scripts/deploy-web-github-pages.ps1`):

```powershell
.\scripts\deploy-landing-github-pages.ps1
```

The full app deploy script also syncs this folder to `deploy-site/landing/`.
